Trident Bot Commands

Simple and powerful commands for server management

Antinuke Commands

  • enable - Enable anti-nuke protection
  • disable - Disable anti-nuke protection
  • antinukelog set/reset/show - Configure anti-nuke logging
  • whitelist - Add user to whitelist
  • whitelisted - Show whitelisted users
  • uwhitelist - Remove user from whitelist
  • extraowner set/reset/list/remove - Manage extra owners

Moderation Commands

  • kick - Kick a user from the server
  • ban - Ban a user from the server
  • unban - Unban a user from the server
  • warn - Issue a warning to a user
  • hide - Hide a channel
  • unhide - Unhide a channel
  • hideall - Hide all channels
  • unhideall - Unhide all channels
  • lock - Lock a channel
  • unlock - Unlock a channel
  • lockall - Lock all channels
  • unlockall - Unlock all channels
  • mute - Mute a user
  • unmute - Unmute a user
  • unmuteall - Unmute all users
  • nick - Change user nickname
  • nuke - Nuke a channel
  • purge - Delete multiple messages
  • role - Manage user roles
  • roleall - Add role to all users

Automod Commands

  • enable - Enable automod
  • disable - Disable automod
  • punishment - Set automod punishment
  • log set/reset/show - Configure automod logging
  • whitelist user/channel/role/category - Whitelist from automod

Ticket System

  • enable - Enable ticket system
  • disable - Disable ticket system
  • help - Show ticket help
  • panelcreate - Create ticket panel
  • panelconfig - Configure ticket panel
  • panellist - List ticket panels
  • paneldelete - Delete ticket panel
  • ticketlist - List all tickets
  • close - Close a ticket
  • delete - Delete a ticket
  • add [user] - Add user to ticket
  • rename - Rename a ticket
  • reopen - Reopen a ticket

Utility Commands

  • avatar - Show user avatar
  • afk - Set AFK status
  • banner - Show user banner
  • list (roles/bots/admins/bans) - List server information
  • membercount - Show member count
  • serverinfo - Show server information
  • roleinfo - Show role information
  • steal - Steal an emoji
  • stealall - Steal all emojis
  • timer - Set a timer
  • userinfo - Show user information
  • snipe - Show deleted messages
  • prefix - Change bot prefix

Bot Info Commands

  • ping - Show bot latency
  • uptime - Show bot uptime
  • support - Get support server link
  • badge - Show user badges
  • premium - Premium information
  • report - Report a bug
  • stats - Show bot statistics
  • invite - Get bot invite link
  • commandinfo - Get command information
  • updates - Show recent updates
  • changelog - Show changelog
  • broadcast - Send broadcast message
  • commands - Show all commands

Message Count Commands

  • messagescount enable - Enable the message count system
  • messagescount disable - Disable the message count system
  • messagescount blacklist add - Add to message count blacklist
  • messagescount blacklist remove - Remove from message count blacklist
  • messagescount blacklist reset - Reset the message count blacklist
  • messagescount blacklist list - View message count blacklist
  • messagescount help - Help for message count commands

Messages Info Commands

  • messages <@user/id> - View message count for a user
  • messages add <@user/id> - Add messages to a user
  • messages remove <@user/id> - Remove messages from a user
  • messages reset <@user/all> - Reset message count

Voice Count Commands

  • voicecount enable - Enable the voice count system
  • voicecount disable - Disable the voice count system
  • voicecount blacklist add - Add to voice count blacklist
  • voicecount blacklist remove - Remove from voice count blacklist
  • voicecount blacklist reset - Reset the voice count blacklist
  • voicecount blacklist list - View voice count blacklist
  • voicecount help - Help for voice count commands

Voice Info Commands

  • voice <@user/id> - View voice activity count for a user
  • voice add <@user/id> - Add voice time to a user
  • voice remove <@user/id> - Remove voice time from a user
  • voice reset <@user/all> - Reset voice count

Leaderboard Commands

  • lbday messages/voice - Show daily top users by messages or voice
  • lbweek messages/voice - Show weekly top users by messages or voice
  • lbmonth messages/voice - Show monthly top users by messages or voice
  • lb messages/voice - Show all-time top users by messages or voice
  • livelb #channel/channelId - Set up live leaderboard in a channel

Leave Message System

  • leave enable - Enable the leave message system
  • leave disable - Disable the leave message system
  • leave message - Edit the leave message content
  • leave help - Display help for the leave message system
  • leave autodel [number] - Set auto-delete time for leave messages
  • leave channel add/remove/list - Manage leave message channels

Boost Message System

  • boostmessage enable - Enable the boost message system
  • boostmessage disable - Disable the boost message system
  • boostmessage message - Edit the boost message content
  • boostmessage help - Display help for the boost message system
  • boostmessage autodel [number] - Set auto-delete time for boost messages
  • boostmessage channel add/remove/list - Manage boost message channels
  • boostmessage dm on/off - Enable or disable DM notifications for boosts
